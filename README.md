## Description

**SAMIMIESS APP** is a small decentralized application (dApp) built with **Solidity** and a **web-based frontend**.  

- Users can **read and write a numeric value**.  
- Users can **read and write a short message**.  
- The frontend connects to the smart contract via **ethers.js**.  
- Wallet integration is done through **MetaMask** on the **Sepolia testnet**.  

This project demonstrates how **smart contracts interact with a web interface** and MetaMask.
